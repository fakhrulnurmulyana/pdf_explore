Ya, file PDF dapat diekstrak ke dalam bentuk script menggunakan pustaka Python. Ada beberapa pustaka yang memungkinkan Anda untuk membaca, mengekstrak, dan memanipulasi file PDF, serta mengonversinya ke dalam format yang lebih terstruktur atau dapat diproses lebih lanjut. Berikut adalah beberapa pustaka Python yang sering digunakan untuk ekstraksi dan manipulasi PDF, serta bagaimana cara menggunakannya:

### 1. **`PyPDF2`**

`PyPDF2` adalah pustaka populer untuk membaca dan memanipulasi file PDF. Meskipun kemampuan ekstraksi teksnya terbatas, ia dapat digunakan untuk mendapatkan informasi dasar dari PDF, seperti metadata, jumlah halaman, dan konten teks.

#### Instalasi
```bash
pip install PyPDF2
```

#### Contoh Penggunaan
```python
import PyPDF2

# Membuka file PDF
with open('file.pdf', 'rb') as file:
    reader = PyPDF2.PdfReader(file)
    
    # Menampilkan jumlah halaman
    print(f"Jumlah halaman: {len(reader.pages)}")
    
    # Menampilkan teks dari halaman pertama
    page = reader.pages[0]
    print(page.extract_text())
```

### 2. **`pdfminer.six`**

`pdfminer.six` adalah pustaka yang lebih kuat untuk ekstraksi teks dan informasi struktur dari PDF, serta mendukung analisis tata letak halaman.

#### Instalasi
```bash
pip install pdfminer.six
```

#### Contoh Penggunaan
```python
from pdfminer.high_level import extract_text

# Menyaring teks dari file PDF
text = extract_text('file.pdf')
print(text)
```

### 3. **`pdfplumber`**

`pdfplumber` adalah pustaka yang lebih canggih untuk mengekstrak teks, tabel, dan elemen lainnya dari PDF dengan lebih detail.

#### Instalasi
```bash
pip install pdfplumber
```

#### Contoh Penggunaan
```python
import pdfplumber

with pdfplumber.open('file.pdf') as pdf:
    first_page = pdf.pages[0]
    print(first_page.extract_text())
    # Ekstraksi tabel jika ada
    for table in first_page.extract_tables():
        for row in table:
            print(row)
```

### 4. **`PyMuPDF` (fitz)**

`PyMuPDF` (dikenal juga sebagai `fitz`) adalah pustaka lain untuk membaca dan menulis file PDF. Ini memberikan akses yang baik untuk mengekstrak teks, gambar, dan metadata dari PDF.

#### Instalasi
```bash
pip install PyMuPDF
```

#### Contoh Penggunaan
```python
import fitz  # PyMuPDF

# Membuka file PDF
doc = fitz.open('file.pdf')

# Menampilkan teks dari halaman pertama
page = doc.load_page(0)
print(page.get_text())

# Menutup dokumen
doc.close()
```

### 5. **Mengonversi PDF ke Format Lain (Opsional)**

Jika Anda ingin mengonversi PDF ke format lain seperti XML atau JSON yang mungkin lebih mudah diproses, Anda dapat menggunakan pustaka tambahan atau alat konversi seperti `pdftohtml` dari Xpdf tools atau alat lain yang serupa.
